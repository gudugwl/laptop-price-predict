# Welcome to Laptop Price Prediction

This system is made by using ensemble model(stacking regeression) where i used RandomForestRegressor, GradientBoostingRegressor and XGBRegressorcosine algorithms for predicing price. Here i have used dataset laptop_data.csv for training model.

## Setup

- First step : create virtual environment

      virtualenv -p python3 venv

- Second step : activate virtual environment

      source venv/bin/activate

- Third step : install package | library from requirements.txt

       pip install -r requirements.txt

- Last step run streamlit server

      streamlit run app.py
